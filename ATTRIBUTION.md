# Attribution Requirements

Thank you for using **HEICtoJPG**.

If you distribute this software or any derivative work, you must provide clear and visible attribution to the original project and authors.

## Required Attribution

Distributions must include:

- Project name: HEICtoJPG
- Author: Martin Campetta
- Link to the original repository (if publicly available)
- A copy of the LICENSE file

## Where Attribution Should Appear

Attribution must be reasonably visible, such as:

- README file
- About dialog
- Documentation
- Credits section
- Source headers
- Distribution package notes

## Example Attribution

Used under the HEICtoJPG Non-Commercial Source-Available License.

## Why This Matters

This project is provided free of charge for the community. Proper attribution:

- Gives credit to contributors
- Helps users find updates
- Supports continued development
