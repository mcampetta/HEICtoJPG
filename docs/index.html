How it works
  1. Launch the app.
  2. Drag‑and‑drop a folder (or click Add Folder) to scan for HEIC files.
  3. Choose settings (quality, delete source, custom output folder, preserve folder structure).
  4. Click Run Conversion and monitor progress.
Key features
  • Multi‑threaded conversion for large folders.
  • Optional custom output directory (preserve or flatten folders).
  • Progress stats (speed, ETA, success/fail).
  • Live preview of latest converted image.
  • Safe delete (moves HEICs to Recycle Bin on success).
  • Optional Explorer right‑click menu for folders.
